Fields ~
{provider} `(optional)` "`(openai)`"
{providers} `(optional)` `(senpai.Config.providers)` see |senpai-config-providers|

Default values:
>lua

    provider = "openai",
    providers = {
      anthropic = {
        model = "claude-3-7-sonnet-20250219"
      },
      openai = {
        model = "gpt-4o"
